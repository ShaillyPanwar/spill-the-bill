# Spill the Bill MVP

## Objective

Allow a group of users to split expenses and calculate balances.

## Features

### Authentication
- Register
- Login
- JWT Authentication

### Groups
- Create Group
- View Groups
- Add Members

### Expenses
- Add Expense
- View Expenses
- Equal Split Only

### Dashboard
- Total You Owe
- Total You Are Owed
- Group Balances

---

## Out of Scope

- OCR Receipt Scanner
- Charts & Analytics
- Email Notifications
- Friend Requests
- Custom Split
